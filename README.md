# Around The U.S.

Aplicación web interactiva desarrollada como parte de mi formación en Desarrollo Web en TripleTen. El proyecto permite al usuario editar su perfil, agregar nuevas tarjetas con imágenes, dar “me gusta”, eliminar tarjetas y visualizar imágenes en una ventana emergente.

## Descripción del proyecto

El objetivo principal de este proyecto fue construir una aplicación web dinámica utilizando HTML, CSS y JavaScript. Durante el desarrollo trabajé con manipulación del DOM, eventos, formularios, validación, clases, módulos y conexión con una API.

Este proyecto representa una etapa importante en mi aprendizaje, ya que combina estructura visual, lógica de programación, organización de código y comunicación con servidor.

## Funcionalidades principales

* Edición de perfil de usuario.
* Agregado de nuevas tarjetas.
* Visualización ampliada de imágenes en popup.
* Funcionalidad de “me gusta”.
* Eliminación de tarjetas.
* Validación de formularios.
* Manipulación dinámica del DOM.
* Organización del código con clases y módulos.
* Integración con API.

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* DOM
* API REST
* Programación orientada a objetos
* Módulos de JavaScript
* Validación de formularios
* Git y GitHub
* Metodología BEM

## Aprendizaje

Con este proyecto reforcé mis habilidades en JavaScript, especialmente en manipulación del DOM, manejo de eventos, validación de formularios y programación orientada a objetos.

También aprendí a organizar mejor el código usando clases y módulos, separando responsabilidades para hacer el proyecto más claro, escalable y mantenible.

# Cafetería Triple Espresso

Landing page desarrollada para una cafetería ficticia como parte de mi formación en Desarrollo Web en TripleTen. El proyecto está enfocado en la construcción de una página estática con estructura semántica, estilos organizados y metodología BEM.

El objetivo de este proyecto fue crear una página web clara y bien estructurada para presentar la cafetería Triple Espresso. La página incluye secciones informativas, videos embebidos, un formulario de reservación y enlaces de contacto.

Durante el desarrollo practiqué la organización de archivos, el uso de HTML semántico, CSS, metodología BEM y buenas prácticas de maquetación.

* Encabezado con navegación y contenido principal.
* Sección de recetas con videos embebidos de YouTube.

* Metodología BEM
* Formularios HTML
* Flexbox
* Normalize.css
* Google Fonts
* Git y GitHub

## Aprendizaje

Este proyecto me ayudó a reforzar la estructura semántica en HTML y la organización de estilos en CSS. También practiqué la metodología BEM para nombrar clases de forma clara y mantener un código más ordenado.

Además, trabajé con formularios, videos embebidos y estructura de secciones para construir una landing page funcional y profesional.
