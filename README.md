## Descripción del proyecto

Este proyecto es una página web interactiva desarrollada como parte del bootcamp de TripleTen.  
La página permite mostrar un perfil de usuario, editar la información del perfil, agregar nuevas tarjetas con imágenes, eliminar tarjetas, dar “me gusta” y abrir una vista ampliada de las imágenes.

El objetivo principal del proyecto fue organizar el código JavaScript usando clases y módulos. Para ello, se crearon las clases `Card` y `FormValidator`, separando responsabilidades y mejorando la estructura del proyecto.
